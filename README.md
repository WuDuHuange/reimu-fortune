#  博丽灵梦的每日一签

一个以东方 Project 中博丽神社为主题的在线求签应用。由博丽神社的巫女——博丽灵梦亲自为你解读运势！

##  功能特色

-  **每日运势占卜** - 点击阴阳玉抽取今日运势
-  **许愿求签** - 可以输入你的心愿，灵梦会针对性地解读
-  **角色扮演** - 由 AI 扮演博丽灵梦，还原其直率、慵懒又贪财的性格
-  **樱花飘落** - 唯美的樱花粒子动画效果
-  **日夜主题** - 根据时间自动切换主题
-  **抽签音效** - 沉浸式的神社铃声音效
-  **历史记录** - 保存最近 30 条抽签记录

##  运势等级

- **大吉** - 运势极佳，万事顺遂
- **中吉** - 运势良好，稳步前进  
- **小吉** - 小有好运，平稳度过
- **吉** - 普通吉利，中规中矩
- **末吉** - 勉强算吉，需多努力
- **凶** - 运势欠佳，小心行事
- **大凶** - 运势低迷，建议多投赛钱

##  技术栈

- **前端**: React + TypeScript + Tailwind CSS
- **动画**: GSAP + Canvas
- **AI**: Google Gemini API
- **部署**: Vercel

##  关于

本项目是一个趣味性的占卜小应用，运势结果仅供娱乐参考。
